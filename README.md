# Event Homepage

This is the official homepage for our upcoming event, showcasing the lineup, music ambiance, and essential links for attendees.

## Features

### Music Playlist
The site includes an ambient music player with the following tracks:

1. **Full Belly - Cera Khin**  
2. **Collision - Audio**  
3. **Push Up - Luninatix Remix**  
4. **Run It Back Twice - GLM**  
5. **Plastikman - Ask Yourself**  

The playlist is available on: 
[Spotify playlist](https://open.spotify.com/playlist/6kHIFozXP9dWXhPlQxhXkz?si=8WNgeImyT2iMn18yPygFaA) 

Users can **play/pause** the music and **skip tracks** while a song is playing.

### Responsive Design
The homepage adapts to all screen sizes:

- **Mobile & Tablet (≤768px)**: Optimized layout for smaller screens.  
- **Desktop (>768px)**: Full-featured experience with expanded visuals.

### Interactive Elements
- Buttons with **hover effects** and subtle cursor changes.  
- Links that indicate clickability without changing the default appearance.  
- Animated **particles** and **audio bars** for a lively experience.

### Event Information
- Lineup page with all performing artists.  
- Ticketing links for easy access.  
- Additional info like schedules, locations, and social media links.

### Modern JavaScript Features
- **YouTube IFrame API** integration for seamless audio playback.  
- Custom playlist controls with reliable skip/play behavior.  
- Optimized animations for smooth performance on all devices.

## Usage

1. Visit the homepage: [Jossgruaz Homepage](https://jossgruaz.github.io).  
2. Use the **sound button** to play or pause ambient music.  
3. Use the **skip button** to move to the next track (only works while music is playing).  
4. Navigate to the **lineup** and **ticketing pages** for full event information.
